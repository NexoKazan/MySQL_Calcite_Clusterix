Строка запуска:

java -jar "MySQL+Calcite+Clusterix.jar" localhost tpch_0 calcite calcite m "SELECT * FROM m.nation"

Параметры:
0 - адрес сервера
1 - имя базы данных
2 - имя пользователя 
3 - пароль (у меня не получилось передать пустой пароль через cmd, поэтому я создал ещё одного пользователя с рут правами: calcite, pass: calcite)
4 - имя схемы для Calcite через него должно быть обращение к таблицам в БД
5 - запрос в кавычках. В поле FROM в имени таблице обязательно должно присутствовать имя схемы из 4го параметра.
